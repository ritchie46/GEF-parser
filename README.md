# gef-parser

Eats .GEF, poops .CSV...

## Convert files
```
gef-parser.exe [filepath.gef] [filepath.csv]
```

## Convert all files in a directory
```
gef-parser.exe [directory] (all .gef files in that directory
                            are converted)
```
